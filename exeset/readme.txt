Platform: WindowsXP, MFC runtime library(MFC71.dll)

Key assign:
  X:Ready
  N:Set
  M:Playing
  S:stop / start

  P:change view point(free/stadium/aibo)

  W:switch wireframe / polygon 
  Z:smooth shading / flat shading

  Q:Save aibo's camera image as ppm file

  I, O, K, L, U:
     add force to the ball

  Ctrl+O:reset ball position
  Ctrl+R:reset aibo position
  
  0-9:motions
  
  A,B, Ctrl+T, Ctrl+B... :-)
